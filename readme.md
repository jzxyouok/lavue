# lavue

学习laravel和vue，用laravel5.3, vue.js 1.0+ 写的demo



## 本地运行

### 克隆代码，安装依赖

```
clone https://github.com/qiuqiuX/lavue.git

cd lavue

composer install

npm install


```

### 数据库设置

修改.env文件的数据库配置

```
APP_URL=your_domain

```


### 运行

```
// 创建数据库，填充数据
php artisan migrate
php artisan db:seed

// 生成前端文件
webpack

```
